DETECT

Deception Tracking Through Eye Cues Technology

Real-time, web-based deception detection using eye-tracking and behavioral analysis

What is DETECT?

DETECT is a web application that analyzes subtle eye and facial movement patterns to surface behavioral signals. It provides a clear, visual summary so non-technical users can understand whether a recorded interaction shows indicators of possible deceptive behavior.

How Recruiters Can Use It

  • Quickly screen recorded interview clips to highlight segments worth reviewing.
  • Complement human judgement with objective visual metrics and timelines.
  • Present consistent, repeatable signals across candidates for fairer comparisons.
View Project on GitHub See Key Features

Overview

DETECT is a real-time, web-based system for analyzing potential deceptive behavior using eye-tracking and behavioral signals. It provides a non-invasive alternative to traditional polygraph-style methods by leveraging computer vision and deterministic signal processing techniques.

The system captures gaze patterns through a webcam or uploaded video, processes eye movement data in real time, and generates a deception likelihood score based on behavioral anomalies.

Installation (optional)

No install needed to understand the project—this section is only for reviewers who want to run the code locally.

macOS & Linux Installation

Prerequisites

Make sure you have the following installed:

  • Git — For cloning the repository. Check if installed: git --version
  • Node.js & npm (optional) — Fallback if you don't want Bun. Check: node --version

Step 1: Install Bun (Recommended)

Bun is a fast JavaScript runtime that replaces Node.js for this project. Install it with one command:

curl -fsSL https://bun.sh/install | bash

After installation, add Bun to your PATH. Restart your terminal or run:

export BUN_INSTALL="$HOME/.bun"
 export PATH="$BUN_INSTALL/bin:$PATH"

Verify installation:

bun --version

Step 2: Clone the Repository

git clone https://github.com/muzamil-dev/detect.js.git
 cd DETECT.js

Step 3: Install Dependencies

With Bun (faster):

bun install

Or with npm (if you prefer):

npm install

Step 4: Start the Development Server

With Bun:

bun run dev

Or with npm:

npm run dev

✓ Your site will be running at http://localhost:4321

Troubleshooting

  • Bun not found: Restart your terminal and verify with bun --version
  • Port 3000 already in use: Check what's running on that port with lsof -i :3000
  • Permission denied on install: Try prefixing with sudo or check folder permissions

Windows Installation

Prerequisites

Make sure you have the following installed:

  • Git for Windows — Download from git-scm.com and install with default options
  • Node.js & npm (optional) — Download from nodejs.org (LTS recommended)

Step 1: Install Bun for Windows

Open PowerShell as Administrator and run:

powershell -c "irm bun.sh/install.ps1 | iex"

If you get an execution policy error, run this first:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Then retry the install command above.

Verify installation by restarting PowerShell and running:

bun --version

Step 2: Clone the Repository

Open Git Bash or PowerShell and run:

git clone https://github.com/yourusername/DETECT.js.git
 cd DETECT.js

Step 3: Install Dependencies

With Bun (faster):

bun install

Or with npm (if you don't have Bun):

npm install

Step 4: Start the Development Server

With Bun:

bun run dev

Or with npm:

npm run dev

✓ Your site will be running at http://localhost:3000

Troubleshooting

  • Bun installation failed: Make sure PowerShell is running as Administrator
  • Git not recognized: Restart PowerShell or use Git Bash instead
  • Port 3000 already in use: Open Task Manager, find the process on port 3000, and kill it
  • 'git' is not recognized: Install Git for Windows and restart your terminal

Prefer npm over Bun?

If you don't want to install Bun, just skip Step 1 and use npm install + npm run dev in the steps above.

Key Features

⚡ Real-Time Gaze Analysis

Processes eye movement data live for immediate feedback with minimal latency.

📹 Non-Invasive Eye Tracking

Uses MediaPipe FaceMesh for facial landmark and iris detection without external hardware.

🔬 Deterministic Detection Engine

Computes deception indicators using gaze dispersion (variance) and directional acceleration.

📊 Personalized Scoring

Min-Max normalization adapts scoring across sessions for user-specific baselines.

🛡️ Robust Signal Processing

Applies Savitzky-Golay filtering and affine transformations for stability under head movement.

🌐 Web-Based Interface

Supports live webcam input and video uploads with real-time visualization and graphs.

Data Flow

  1. Video input is captured via webcam or uploaded file.
  2. Astro frontend processes frames using MediaPipe FaceMesh.
  3. Eye landmarks are converted into gaze coordinate data.
  4. Data is streamed to backend via WebSockets.
  5. Go backend applies filtering and statistical analysis.
  6. Deception score is computed using deterministic rules.
  7. Results are stored and visualized in real time.
